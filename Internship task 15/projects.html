Task Manager App

A comprehensive full-stack task management system built to streamline daily productivity. The frontend uses React.js for a smooth user experience while the backend is powered by Node.js with Express.js and MongoDB for robust data handling.

  • Create, update, delete, and filter tasks with ease
  • User authentication and authorization implemented securely with JWT
  • Visual progress tracking with dynamic status indicators
  • Mobile-friendly responsive design ensuring usability across all devices

Key Challenges: Managing state and asynchronous API calls in React, designing secure authentication flows, optimizing database queries for performance.

Learnings: Improved skills in RESTful API development, advanced React hooks, JWT-based security, and responsive UI/UX design.

Technologies: React.js, Node.js, Express.js, MongoDB, JWT, CSS3
